Exportar Base de Datos

Descarga un archivo .sql con todos los datos del sistema

El archivo incluye todos los registros de: usuarios, clientes, productos, ventas, pedidos y configuracion. Guarda este archivo en un lugar seguro para poder restaurar el sistema en caso de falla.

Restaurar Respaldo

Importa un archivo .sql generado por este sistema

Importante: La restauracion ejecuta sentencias INSERT IGNORE sobre los datos existentes. No elimina registros actuales. Solo sube informacion nueva o que no exista en la BD actual. Solo acepta archivos .sql generados por ROSS MILLE.
